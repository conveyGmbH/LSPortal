LeadSuccess Portal - Masterdata

Welcome to the new portal application for LeadSuccess - Masterdata.

This application is only for yearly-rent customers. The application offers 2 main functions:

  1. Create new users for all roles
  2. Move one user from their current event to another, without the need for the user to log-out from the APP on his/her device.

Function 1 - Create New Users

This application allows you to create new user with different roles. The following roles are available:

  1. APP-User: a user who has access to the APP. No access to the LeadSuccess Portal.
  2. Admin: access to all functions of the LeadSuccess Portal, but only for the event the user is assigned to. No possibility to create new users, change event or create new events.
  3. Customer-Admin: access to all functions of the LeadSuccess Portal, access to all events, possibility to create new users and new events.
  4. Infodesk: has only access to the Info-Desk application for the event assigned to this user.
  5. Revision: has only access to the startpage, contacts and events, dashboard and option application for the event assigned to this user.
  6. Reporting: has only access to the startpage, contacts and events, dashboard, exports and option application for the event assigned to this user.

How to create a new user:

  1. Select the event for which you want to create the user.
  2. Click on "Create New Employee" on the top right
  3. Enter name, surname, username and password (twice the same password).
  4. Select the role, then confirm by click on the "Save Changes".

You can reassign this or any user in the Masterdata application at any time. To do this, please read the next chapter (Function 2)

Function 2 - Reassign Users to Other Events

In this application you can:

  1. Reassign users from their current event and assigned them to another event.
  2. Deactivate users to free one or more slots in one event for other users.

To reassign a user from the event they are current assigned, please do as follows:

  1. Select the event the user is currently in.
  2. Select the user you want to reassign.
  3. Click on the "Activate" button next to the event you want the users reassigned to.

After reassigning a user by clicking on activate, an hourglass symbol will be shown next to the user, until the user accepts the reassign in the APP. When he accepts (he can postpone the reassignment in the APP for 5 minutes at a time), the hourglass symbol will disappear.

During this time, while the hourglass symbol is active, you can cancel the reassignment by clicking on the "Renew Target" button.

To deactivate a user, to free one or more slots in one event for other users, please do as follows:

  1. Select the event the user is currently in.
  2. Select the user you want to deactivate.
  3. Click on the "Deactivate" button next to the event the user is currently assigned to.

Deactivated users have an "X" symbol next to them.
